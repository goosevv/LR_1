LR_1 My first repository

Що таке Git? це розподілена система керування версіями файлів

В якому статусі можуть перебувати файли при роботі з Git? збережений у коміті (commited), змінений (modified), індексований (staged)

Для чого використовується  команда add? для додання файла для коміту

Для чого використовується  команда status? для визначення статусу каталогу

Для чого використовується  команда commit? ця команда бере данні які ви добавили завдяки git add і закомічуе їх (зберігає)

Для чого використовується  команда init? щоб включити сам Git для репозиторію

Що необхідно писати в коментарях комітів? які зміни ви зробили або щось вказати, що вважаете важливим, що захочете

Як переглянути список комітів? команда git log

Для чого використовується  команда checkout? щоб переходити між різними гілками
